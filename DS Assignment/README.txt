HOW TO RUN

1. Start wamp
2. Start mysql console (no password).
3. Copy contents of data.sql file and run in mysql console.
4. Open project in eclipse
5. In the RMI folder, navigate to ie.gmit.sw.RMI.CarHireDBSetup and run as java application.
6. Next, right click on dsAssignment folder and run  as-> run on server 
7. Next, right click on WebClient folder and run as -> run as spring boot project. (run at http://localhost:8081/index.html)
8. Next, in the Deskstop client, navigate to the DesktopHTTPClient and run as java application.
