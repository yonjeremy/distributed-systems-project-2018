HOW TO RUN

1. Open project in eclipse
2. In the RMI folder, navigate to ie.gmit.sw.RMI.CarHireDBSetup and run as java application.
3. Next, right click on dsAssignment folder and run  as-> run on server 
4. Next, right click on WebClient folder and run as -> run as spring boot project.
5. Next, in the Deskstop client, navigate to the DesktopHTTPClient and run as java application.
